# 人物介绍：

# 朱长留（夫诸帝）：

  喜欢的人被杀，自己却只能眼睁睁被看着，被人所救，假死十五年

# 安迪（英招帝）：

  纯正的玡族后人，杀伐果断的战争武器

# 苏心怡：

  朱 xx 喜欢的人，后来嫁给了别人，实际身份是财团组织的千金，潜入主角身边，想刺杀主角

# 布・xx・xx：

  财团打手，但却是被人利用，他的身份也待揭晓

# 博士：

  反财团组织的军师，后组织事变，带着重要的东西隐居，逃过被杀一劫。十五年后决战起到关键作用

# 苏菲亚（冰夷神）：

  从小被传销组织所骗，被人封为恶魔女王

# 苏丽娜：

  妹妹，野心十足

  五十年前，神之怒降临，诞生了魔族（发色为白，眼色为铜）和玡族（脸上有疤痕缺陷），二者备受歧视，在背后操控是当时的财团 ——XXX。后因族人的不断斗争，为自己争取到了主权，过上了和正常人有着同等权利的生活。

  十五年前，玡族事变，麟威帝逝世，临终前其将自己的力量化为两颗晶石分别给了自己的两个儿子，一人不幸被抓，一人不知所踪。玡族也因此沦落为财团的工具，族人保守欺辱。

  如今，玡族首领成了财团的傀儡，某些因饱受欺辱的玡族人成立了反财团组织，其中也有伸张正义的正常人，包括主角在内的七人组成了反歧视小队。后因一部分人叛变，组织分崩离析，被财团吞并，并改名为 xxx。于此同时，魔族女王下落不明，年仅七岁的苏菲亚当上了女王。

  后来魔女力量觉醒，成为了改变世界的关键，一场争夺魔女的战役就此打响
