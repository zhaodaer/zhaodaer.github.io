  • # Just about me:

  • sunonzj，男，一个刚毕业的 java 小菜鸡！虽然是小菜鸡，但是却有一个成为大牛的心，所以想用这个博客记录下成长过程。因为最近时间和精力都比较充裕，就搭建了这个博客。一开始想做自己的博客是在大一，有一次刷知乎的时候无意点进了 stormzhang 的博客链接，学到了很多，也埋下了自己建站的种子。那时候只是觉得用博客记录学习过程很有意义，所以就想做个自己的网站。只是当时毕竟刚接触又太懒感觉做不出来就拖到现在。没错，居然拖到现在，都毕业了。后来又发现了博客的一些好处，所以还是做出来了。这不只是一个技术博客，更像一个我的生活记录册！

  • 虽然会有些晚，因为错过了大学这么美好的时光。很多东西没有记录到博客中了。那时候对开发有了从 0 到一的认识。也是对未来最迷茫的时候。现在加上实习也快工作一年了这一年对自己有了更清楚的认识，对未来也有了自己的期待。 希望能够记录未来的美好生活。一个小菜鸡的蜕变之路！希望无论以后会经历啥变成啥样的人，都能够把博客一直更新下去，算是立个 flag 吧，所以感兴趣的话可以多多访问哦。 建立这个网站的初衷一是为了分享生活、记录成长。二是可以发表一些自己的感悟、和学习体会。希望也能够帮助到一些人。或许你在网上冲浪的时候刚好看到这个网站，而这个网站又刚好有你需要的答案。人生就是一个得与失的过程，虽然不知道未来是什么样，但现在我会坚定自己的初心，希望能够学到更多的东西、在 java… 中激发了兴趣，然后越走越远…

  • 这个网站也算是费了心思的，从前台到后台，从前端到后端，从项目设计到需求分析，从构想到功能实现，从数据库设计到服务器配置，很多都是从 0 到一，就做这个网站也学到了很多，大大小小二十几页面的代码都是利用休息时间一行一行敲的。我看了很多人的博客，学习到了很多东西。现在上班闲时也常逛几个博客，看看他们的文章，希望我也能够越来越棒吧！学习是一个枯燥的过程，需要不断克服路途中遇到的困难，要始终坚持心中的信念。最后一句话：“不必太纠结于当下，也不必太忧虑未来，当你经历过一些事情的时候，眼前的风景已经和从前不一样了” ————2018 年 7 月 26 日

  • 好想删掉上面的，想想还是算了，等再过几个月把关于我这个页面 换一下吧，弄个新的页面出来，这个就当老版本也保存着吧。毕竟我是个喜欢怀旧的人～————2019.5.10
